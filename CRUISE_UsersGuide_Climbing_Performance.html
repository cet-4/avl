Climbing Performance

The Climbing Performance Task includes two calculation possibilities:

  • Maximum ascending gradients that can be surmounted will be subjected to a stationary computation for all possible velocities.
  • Maximum ascending gradients at which a starting is still possible with a desired starting acceleration will be acquired for each gear step.
Figure 1. Climbing Performance Window

The climbing performance depends on:

  • Velocity Measuring Points (optional)
  • Course
  • Driver

The reference axis is the vehicle velocity.

Climbing Performance

Run

Part task is calculated.

Do not run

Part task is not calculated.

Output of Text Report

By default, the output of template based text outputs is activated. To save calculation time, it can be suppressed.

Output of MS-Exports

This is deactivated by default. When activated, the MS-Exports components write their Data Bus input data into an excel file. The output has to be activated also in every MS-Exports component.

Hot Start - steady state

The temperature models of the engine, respectively the exhaust gas system is deactivated.

Cold Start

The temperature models of the engine, respectively the exhaust gas system is deactivated. The ambient temperature is used as the start temperature of the engine and the exhaust system.

Driving Resistance

One of five vehicle resistance models should be determined for calculation. The same model should also be activated in the Vehicle's properties window.

Physical

The driving resistance is defined by the physical values (drag coefficient, rolling resistance).

Function with Reference Vehicle

The driving resistance is defined by three parameters. The defined values are converted from the reference vehicle to the actual one.

Characteristic with Reference Vehicle

The driving resistance is defined in a characteristic as function of the vehicle velocity. The defined values are converted from the reference vehicle to the actual one.

Function without Reference Vehicle

The driving resistance is defined by three parameters. The defined values are used without a reference vehicle.

Characteristic without Reference Vehicle

The driving resistance is defined in a characteristic as function of the vehicle velocity. The defined values are used without a reference vehicle.

Stationary S / Stationary 1 / Stationary 2

The user can select which solver is used for this task. For special drive train configurations only stationary S is possible. For general drive train configurations the user can select between two solvers (stationary 1 and stationary 2). Stationary 2 should be activated first in the settings window.

Starting from Rest with Start Acceleration

When activated, it is possible to calculate the take off ascending gradient at a defined take-off acceleration for the first gear in addition to the maximum ascending gradient that can be surmounted at each engine speed.

Partial Load with Load Position

When activated, the maximum load position for the task can be defined.

Shifting

According to Velocity

The gear box control, respectively the driver adjusts the gears according to the vehicle velocity.

According to Speed

The gear box control, respectively the driver adjusts the gears according to the engine speed.

Tiptronic

With automatic transmission, the gears are controlled by the gear box control, respectively the gear box program. Since the individual gears cannot be controlled directly, there is only one result line for each driving level. Tiptronic is used to investigate each of the gear steps. When activated, the driver is in control and can shift the individual gears. This is insignificant with a manual transmission.

Kickdown

When activated the gears will be shifted according to the Kickdown Characteristic which can be defined in the Gear Box Program. Kickdown means that the load signal is 100%. This is insignificant with a manual transmission.

Shift gears

Gear Box Control

For automatic cockpits it can be determined that the gear box control controls the gear shifting process.

Gear Shifting Program

For automatic cockpits it can be determined that the gear box program controls the gear shifting process.

Without slip

Slip of the wheel is not considered.

Without slip - limited

Slip of the wheel is not allowed. The maximum longitudinal force is limited with the friction coefficients of the road and the tire.

With slip

Slip of the wheel is considered.

Measuring points

Velocity active

The velocity measuring points are activated.

Inclination at Speed

To obtain the ascending gradient for all speeds for special engine speeds, press the ascending button and enter the desired engine speeds.

Driver and Course

The driver file and the course file can be loaded form the general directory.
Load State

The calculation can be run with the following load states:

Empty: the weight of the vehicle equals the Curb weight

Half: the weight of the vehicle equals the middle of the Curb and the Gross weight

Full: the weight of the vehicle equals the Gross weight

Vehicle

More than one load can be selected. In this case, the calculation will be done once for every load state.

An additional mass for the load state empty can be defined so that the calculation can be run with every different weight of the vehicle.

Constant Additional Load (Abs)

A constant load has to be defined which is added to the load of the empty vehicle.

Constant Additional Load (Rel)

A percentage of (Gross weight - Curb weight) has to be defined which is added to the load of the empty vehicle.

Constant Number of Passengers (Abs)

A number of passengers and average passenger weight have to be defined. Overall passenger weight is then added to the load of the empty vehicle.

Constant Additional Volume (Abs)

A volume of the cargo and its density have to be defined. Overall cargo weight is than added to the load of the empty vehicle.

Trailer

More than one load can be selected. In this case, the calculation will be done once for every load state.

An additional mass for the load state empty can be defined so that the calculation can be run with every different weight of the trailer.

Constant Additional Load (Abs)

A constant load has to be defined which is added to the load of the empty trailer.

Constant Additional Load (Rel)

A percentage of (Gross weight - Curb weight) has to be defined which is added to the load of the empty trailer.

Constant Additional Volume (Abs)

A volume of the cargo and its density have to be defined. Overall cargo weight is than added to the load of the empty trailer.

Virtual Trailer

The task can also be calculated with a 'virtual trailer'. There is an additional pulling force which acts in parallel to the surface of the roadway. With the virtual trailer the effects of a trailer on the climbing performance can be calculated without modeling it in the Vehicle Modeler.

Caution! The setup of the virtual trailer is independent from the fact that a trailer is modeled in the Vehicle Modeler. In extreme cases, two trailers can be considered.
