Application of the Program

The computation of the vehicle performance includes:

  • Constant Drive
  • Climbing Performance
  • Full Load Acceleration
  • Maximum Traction Force
  • Brake/Coast/Thrust

The simulation of the real driving conditions while driving along a road can be done with the following calculation tasks:

  • Cycle Run
  • Cruising

The consumption and emissions for a specific driving profile are the main results of these computations.

System Analysis simulation is used to analyze the parts of the driveline or components in the sub-system.

More than one task can be selected for one program run except in SAM (System Analyse Mode) which is a stand alone task folder with System Analysis task only.
