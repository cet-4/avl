Calculation Results

There are two types of files for the calculation results:

  • Result Files generated once for the whole project
  • Result Files generated for every calculation task of the project

These files are also stored in different directories.

The result files generated for the whole project are located in the messages directory of the corresponding version, respectively project directory e.g. 
\projects\car_1\ver_0001\messages\

These files are:

Cruise.log

Contains information about the calculation, the calculation time, if extrapolations had to be made, if errors had occurred, etc.

Read.log

Contains information about which input data were read from the Data File.

Notread.log

Contains information which input data were not read from the Data File.

Results.log

Contains results of the calculation in text form. This file can be freely defined by the user via text templates (refer to Chapter 6.4 Creating Report Sheets). If this file does not appear, there are two possibilities:

  • The Output of Standard Texts is deactivated
  • The calculation has been canceled due to an error
The result files generated for every calculation task are located separately for every calculation task in the corresponding task directory e.g. 
\projects\car_1\ver_0001\folder.001\task.001\

These files are named according to their results e.g.

m__1___3.g_1m result output of a mechanical component

c_61__16.g_1 c result output of a control component

t__1___4.g_1t result output of a calculation task

d__1___2.pp1d result output of a diagram template

The filename description is different for the components, calculation task and templates.
