Properties

Output

If activated a result output for this component is printed.

Idle Speed

Constant

The definition of the idle speed reference is constant over the complete simulation.

From Data Bus

The definition of the idle speed reference varies during simulation.

Launch Speed

Constant

The launch speed target is maintained constant over the complete simulation.

From Characteristic

The launch speed target varies according to a characteristic where the speed values are changing in relation with the engine desired torque. Usually the higher the torque request, the higher the reference launch speed.

From Data Bus

In the case of external correlation between some generic parameters and launch speed, this option gives the opportunity to define it via data bus.

Launch Control PI Parameters

Constant

The control of the launch speed is realized by means of PI control. With this option the PI parameter set will remain constant over the complete simulation.

From Data Bus

The PI parameters can be modified during the simulation and according to any external correlation.

Shift Control PI Parameters

Constant

The control of the shift procedure is performed by means of PI control. With this option the PI parameter set will remain constant over the complete simulation.

From Data Bus

The PI parameters can be modified during the simulation and according to any external correlation.
