Buck Converter

In the step down converter configuration the current flows from the higher network (electrical system) to the lower network.

In the following part the basic equation of power without power loss is described:

\[ P_{DC,Buck,\,HV,act} =P_{DC\,,Buck,LV,\,act} \]
(1)
\[ {U}_{{DC,}Buck\,,{HV,}\,{act}} \cdot I_{DC,\,Buck,HV,\,act} =U_{DC,Buck\,,\,LV,\,act} \cdot I_{DC,Buck,\,LV,\,\,act} \]
(2)

The maximum current from the higher net is defined according to the following equation

\[ {U}_{{DC,}Buck{,}\,{HV,}\,{max}} \cdot I_{DC,Buck,\,HV,\,\max } =U_{DC,Buck,\,\,LV,\,\max } \cdot I_{DC,Buck,\,\,LV,\,\,\max } =P_{DC,\,\max } \]
(3)
\[ I_{DC,Buck,\,HV,\,\max } =\dfrac{P_{DC,\,\max } }{U_{DC,Buck,\,HV,\,\max } } \]
(4)
Figure 1. Scheme of Buck Converter

The maximum current flowing into the lower net is defined as:

\[ I_{DC,Buck,\,LV,\,\max } =\cfrac{P_{DC,\,\max } }{U_{DC,Buck,\,LV,\,\max } } \]
(5)

when the power loss is taken into consideration the basic equation will be defined as:

\[ P_{DC,Buck,\,HV,act} =P_{DC,Buck\,,LV,\,act} +P_{DC,Buck,\,loss} \]
(6)

The actual power loss in the DC/DC Converter is total loss of power of electronic elements such as inductance, internal resistance, diodes, etc.

The power losses of all complex processes within the DC/DC converter are described here with two factors:

Power of converter \( P_{DC} \) and efficiency of converter as a constant or a characteristic curve

\[ \eta_{DC} (P_{DC,\,act} ) \]
(7)

The efficiency is defined in the following equation:

\[ \eta_{DC,\,\,act} =\cfrac{P_{DC,Buck,\,LV,\,\,act} }{P_{DC,Buck,\,HV,\,act} } \]
(8)
\[ I_{DC,Buck,\,LV,\,act} =\eta_{DC,\,act} \cdot \cfrac{U_{DC,Buck,\,\,HV,\,act} \cdot I_{DC,Buck,\,HV,\,act} }{U_{DC,Buck,\,LV,\,act} } \]
(9)

the power loss is defined as the difference between input power and output power:

\[ P_{DC,Buck,\,Loss,\,act} =(1-\eta_{DC,\,act} )\cdot U_{DC,Buck,\,HV,\,\,act} \cdot I_{DC,Buck,\,HV,\,act} \]
(10)

Theoretical actual Maximum Values on the both voltage Nets:

Maximum current from the higher net ( Eq.4).

maximum current into the lower net:

\[ I_{DC,Buck,\,LV,\,\max } =\eta_{DC} \cdot \cfrac{U_{DC,Buck,\,\,HV,\,\max } \cdot I_{DC,Buck,\,HV,\,\max } }{U_{DC,Buck,\,LV,\,\max } } \]
(11)

the theoretical actual maximum power into the lower net is therefore defined as:

\[ P_{DC,Buck,\,LV,\max ,\,act} =I_{DC,Buck,\,LV,\,\max } \cdot U_{DC,Buck,\,LV,\,act} \]
(12)

This actual maximum power into lower net must be the internal limit of the power from the higher net:

\[ P_{DC,Buck,\,HV,\,act} <\,=P_{DC,\,LV,\,\max ,\,act} \cdot \left( {\eta _{DC} } \right)^{-1} \]
(13)

the actual maximum power from the higher net is therefore defined as:

\[ P_{DC,Buck,\,HV,\,\max ,\,act} \,=P_{DC,Buck,\,LV,\,\max ,\,act} \cdot \left( {\eta_{DC} } \right)^{-1} \]
(14)

The current flow controller

The current flow will be cut off when:

1. \( U_{DC,Buck,\,HV,\,act} <\,=U_{DC,\,Buck,\,HV,LT} \)

2. \( U_{DC,Buck,\,LV,\,act} >\,=U_{DC,\,Buck,\,LV,\,UT} \)

The current flow will be cut in again when:

\( U_{DC,Buck,\,LV,\,act} <U_{DC,\,Buck,\,LV,LT} \) and in the meantime the \( U_{DC,Buck,\,HV,\,act} \)

has exceeded the activation threshold \( U_{DC,\,Buck,\,HV,\,UT} \)

The current control with regulation value

The actual current from the higher net can also be controlled by a regulation value from data bus. That means the regulation value (\( \alpha_{DC,Buck} \)) regulates (reduces) the current of the higher net:

\[ I_{DC,Buck,HV,\,regulated} =\alpha_{DC,\,Buck} \cdot I_{DC,Buck,\,HV,\max } \]
(15)

\( \alpha_{DC,\,Buck} \) is a value in \( \left[ {0......1} \right] \)

In order to limit the current, which will be delivered into the lower net, the actual current will be limited by the regulation current:

\[ I_{DC,Buck,\,HV,\,act} <\,=\,I_{DC,Buck;HV,regulated} \]
(16)

when the regulation value is equal to zero there is no current flow into the lower net. When the regulation value is not connected in the data bus then there is no regulation of current and upper limit is as before:

\[ I_{DC,Buck,\,HV,\max } \]
(17)
